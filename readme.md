A testbed for new features, built with Roots

- Jeet

## Powered by:

- Roots Static Site Generator
- - Jade
- - Stylus
- - - Axis
- - - Jeet



### Setup

- make sure [node.js](http://nodejs.org) and [roots](http://roots.cx) are installed
- clone this repo down and `cd` into the folder
- run `npm install`
- run `roots watch`
